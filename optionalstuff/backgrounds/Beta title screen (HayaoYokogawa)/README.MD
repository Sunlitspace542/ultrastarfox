# Note
For accuracy, if you're using this comment out this line in `SF\MAPS\TITLE.ASM`:

`	mapobj	0,20,20,70,my_demo,tit_istrat`

These graphics overwrite and use the US Revision 1/2 title screen palette (set `titlelogo` to 2 in `SF\CONFIG\GRAPHICS.INC`).

You may need to clean the build first for the graphics to compress.
